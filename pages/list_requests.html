Request ​

  • HTTP Method: GET
  • Content Type: application/json
  • URL: [BASE_URL]/v1/third-app/activities/[activity:uuid]/demandes

Request example ​

js
axios.defaults.headers.common['Authorization'] = ACCESS_TOKEN;

axios.get('BASE_URL/v1/third-app/activities/activity:uuid/demandes', 
       [
          headers: {
            'Content-type': 'application/json',
          }
      ],
})
