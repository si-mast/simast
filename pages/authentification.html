Oauth Token ​

Request ​

  • HTTP Method: POST
  • Content Type: application/json
  • URL: [BASE_URL]/oauth/token

Parameters ​

PropertyTypeRequiredDescriptionvalue
grant_typeStringtrueThe authorization grant typeclient_credentials
client_idStringtrueThe app id sended by mail
secretStringtrueThe app key generate from Simast Portal

Request example ​

js
axios.post(BASE_URL/oauth/token', 
      {
        grant_type: 'client_credentials,
        client_id: 'string',
        secret : string
      },
       [
          headers: {
            'Content-type': 'application/json',
          }
      ],
})

Response example ​

JSON
 {
   "token": "string",
   "token_type": "string",
   "expires_at": "string"
 }
