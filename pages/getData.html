Request ​

Option 1 with search Features ​

  • HTTP Method: POST
  • Content Type: application/json
  • URL: [BASE_URL]/v1/third-app/demandes/[demande:details:uuid]/data/search

body ​

PropertyTypeRequiredDescription
searchObjectfalseSearch data fields 'code', chef_nom, chef_prenom, 'membres.nom', 'membres.prenom'
json
{
  "search" : {
      "value" : ""
    }
}

params ​

NameValue
page0Current page
limit0Limit /pages

Request example ​

js
axios.defaults.headers.common['Authorization'] = ACCESS_TOKEN;
js
axios.post('BASE_URL/v1/third-app/demandes/demande:details:uuid/data/search', 
      {
        "search" : {
            "value" : ""
        }
      },
       {
          params: {
            page: 0,
            limit: 0
          }
          headers: {
            'Content-type': 'application/json',
          }
      },
})
