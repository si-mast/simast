List Activities ​

Request ​

  • HTTP Method: GET
  • Content Type: application/json
  • URL: [BASE_URL]/v1/projects/[project:uuid]/activities

Request example ​

js
axios.defaults.headers.common['Authorization'] = ACCESS_TOKEN;

axios.get('BASE_URL/v1/projects/project:uuid/activities', 
       [
          headers: {
            'Content-type': 'application/json',
          }
      ],
})
